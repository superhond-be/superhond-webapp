Deze handleiding legt stap voor stap uit hoe je het SETUP_TOKEN instelt in Render en hoe je je eerste admin-accounts aanmaakt.

1. Ga naar je service in Render

  1. Log in op Render.
  2. Klik op je service superhond-webapp.

2. Open het menu Environment

In de linkerkolom zie je opties zoals Events, Settings en Environment. Klik op Environment.

3. Voeg de variabele toe

Klik op Add Environment Variable en vul in:

  • Key: SETUP_TOKEN
  • Value: kies een geheime code, bv. Superhond2024!

Klik daarna op Save Changes. Render start de app opnieuw op.

4. Registreer admins

  1. Ga naar /admin-register.html.
  2. Vul bij Setup token exact de waarde in die je net in Render zette.
  3. Vul Naam, E-mail en Wachtwoord in.
  4. Klik Aanmaken.
  • De eerste gebruiker wordt automatisch superadmin (Sofie).
  • De tweede gebruiker wordt automatisch admin (Paul).

5. Na registratie

  • Inloggen kan altijd via /admin-login.html (e-mail + wachtwoord).
  • Nieuwe externen kunnen door Sofie (superadmin) toegevoegd worden via Dashboard ‚Üí Admin gebruikers.
  • Als de eerste accounts klaar zijn, verwijder of wijzig je de SETUP_TOKEN variabele in Render voor extra veiligheid.
